PRIMEIROS PASSOS PARA TI / UNIDADE TÉCNICA:
1. Certifique-se de que o banco "bd_stmi.MDB" tem permissão de escrita para a equipe.
2. No App, a seção "Integração MDB" agora aponta diretamente para este local.

COMO SINCRONIZAR OS DADOS DO APP COM O ACCESS:

MÉTODO A: TABELAS VINCULADAS (RECOMENDADO)
------------------------------------------
1. No App STMI, clique em "Exportar CSV para Access".
2. Salve o arquivo "DADOS_PARA_ACCESS.csv" na mesma pasta do seu .MDB.
3. No Microsoft Access, vá em: Dados Externos > Novo Banco de Dados > Do Arquivo > Texto.
4. Selecione "Vincular à origem de dados criando uma tabela vinculada".
5. Selecione o CSV exportado.
6. Pronto! Agora, toda vez que você exportar o CSV do App e sobrescrever o antigo,
   o Access atualizará as consultas e relatórios automaticamente.

MÉTODO B: EXECUÇÃO DE SCRIPT SQL
--------------------------------
1. Gere o arquivo .SQL no App.
2. No Access, crie uma "Consulta em Modo SQL" (Query Design > SQL View).
3. Cole o conteúdo do script e execute.
   *Nota: O script gerado usa o dialeto ANSI compatível com macros do Access.*

VANTAGENS DESTA IMPLEMENTAÇÃO:
- Não requer instalação de drivers ODBC complexos no navegador.
- Funciona 100% offline se a rede cair (os dados ficam no cache do App).
- Integridade total com o caminho \\10.9.0.211 fornecido.
